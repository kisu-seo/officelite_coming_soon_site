## My process

### Built with

- **Vanilla JavaScript** — Implements custom interactive components including a keyboard-accessible dropdown listbox, live form validation with dynamic error feedback, and a persistent countdown timer synced with Local Storage.
- **Tailwind CSS 4** — Handles all styling via utility classes with a custom `@theme` design token system (colors, typography presets, spacing) and responsive breakpoints optimized for desktop and mobile layouts.
- **Vite 6** — Serves as the frontend build tool, offering a fast development server (HMR) and optimized multi-page production builds via `@tailwindcss/vite`.
- **Semantic HTML5 & Accessibility (A11y)** — Built with semantic tags (`<main>`, `<section>`, `<header>`, `<footer>`), ARIA attributes (`aria-label`, `aria-expanded`, `aria-haspopup`, `role="listbox"`, `role="option"`), and keyboard-accessible interactive controls throughout.
